ABS CCG

A place to create silly cards and trade them with your coworkers

Stuff to know about this site

A lot of time went into this project, and development is ongoing. It's still never going to be perfect. Steps were taken to help make it secure and stable, but there are probably issues that were missed or forgotten about.

Everything has been designed to work when the user navigates around in their browser by clicking. You can look at the URLs and make good guesses for how to get around, but you might end up somewhere you didn't expect.

Steps were taken to prevent forged POST requests from messing things up, but there could still be loopholes. For the sake of everyone's enjoyment, please don't try to forge requests to mess with other users. We're here to have fun.

User accounts

The first step to enjoying this wonderful site is making an account. All you need is a username and password. Your username is how other users will see the cards you've collected and initiate trades with you. There is currently no censoring on names. Use common sense.

To sign out. Delete your cookies for this site. Heh

Security

SSL is used, and passwords are never stored in plaintext on the server. Common sense should still tell you not to use a password that is shared with other accounts you care about though.

Many pages will check if you're logged in before allowing your browser to render anything. If you're not logged in, an automatic redirect will occur, prompting for you to do so. After successful authenticatin, the browser returns to the page you initially tried to visit.

Changing your name/password

There is not currently a way for a user to change his name or password. Things could be manually edited at a database console, but not easily. This feature could be added later if there is enough demand for it.

Cards

Anyone with an account can create cards, and these cards will be available for anyone to use. Let's try not to make cards that are too easy or hard to get. e.g."Literally died" or "Inhaled"

If you make a card in error, you can delete it. Please don't delete cards that are already in circulation. The system allows it, but it creates issues that need to be cleaned up by hand.

Just like usernames, no card content is censored. Use common sense.

Trades

The bulk of the upcoming changes to this site will revolve around trades. There are known bugs right now that allow trades to process with lines that are no longer valid (e.g. User has deleted a card from their collection, but the card is still part of a trade). The system will currently ignore lines that contain errors but let the rest of the trade go through, so it's possible to rip someone off and get a card while giving nothing.

While this could happen by accident, it can mostly be avoided by just making sure you only have one active trade at a time. We're not here to scam each other, so just be careful.
